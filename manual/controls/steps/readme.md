引导用户按照流程完成任务。

## API

### `Steps`

| 属性名 | 值类型/可选值 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `direction` | `'horizontal' | 'vertical'` | `'horizontal'` | 步骤条铺展方向 |
| `activeFlag` | `number` | `0` | 指定当前步骤，从 `0` 开始记数 |
{:.table.table-bordered}

### `Step`

| 属性名 | 值类型/可选值 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `title` | `string` | - | 步骤标题 |
| `description` | `string` | - | 步骤描述 |
{:.table.table-bordered}
